This project is a Flutter-based To-Do List application designed as part of an internship task to demonstrate state management, local data persistence, and dynamic UI building. The app allows users to manage their daily tasks efficiently with the ability to add, edit, delete, and mark tasks as complete.

Features:-
Add Tasks,Users can add new tasks via a text input field.,Edit Tasks,Tasks can be edited after creation by clicking the edit icon.,Delete Tasks,Tasks can be deleted individually using the delete icon.
Mark Tasks as Done/Undone
Toggle task completion status using a check icon.Completed tasks show line-through text and a green check.Persistent Storage
Tasks are saved locally using SharedPreferences.Data is retained even after the app is closed and reopened.User-Friendly UI
Clean and responsive layout using ListView and Card widgets.
Visual cues for completed tasks and action icons.

Technologies Used:-
Flutter & Dart,Widgets: Scaffold, ListView, Card, TextField, IconButton
Local Storage: SharedPreferences
State Management: setState
